# Sprachbaum

Structured German learning platform (CEFR A1–C1) with spaced repetition,
real exercises, and detailed progress analytics. Built to actually learn
the language — not to farm streaks.

> 🚧 In active development. MVP scope: complete A1 level.

## Stack

Next.js 15 · NestJS · PostgreSQL · Prisma · Redis · Docker
